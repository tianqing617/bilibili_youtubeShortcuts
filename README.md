### 一、介绍

为B站（bilibili.com）实现与youtube相同的播放器快捷键功能。  
| 功能 | 快捷键 |
|----------|--------|
| 全屏 | f |
| 网页全屏 | w |
| 剧场模式 | t |
| 静音 | m |
| 弹幕开关 | c |
| 循环播放 | r |

> 后续计划：自动播放、自定义快捷键...
